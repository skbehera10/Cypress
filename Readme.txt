
Please run below commands to setup and open cypress

For Cypress Olderversion(<10):-
1. Download from github
2. Open folder Xe.com-Cypress9.7.0 in the Editor
3. Execute -
For Cypress Olderversion(<10)
 	npm install cypress@9.7.0 --save-dev
	npx cypress open

For Cypress Current Version:-
1. Download from github
2. Open folder Xe.com-CurrentVersion in the Editor
3. Execute -

 	npm install cypress --save-dev
	npx cypress open
